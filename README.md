VRoid Reference Overlay Tool (VRoid 参考图悬浮工具)English | 中文<a name="chinese"></a>🇨🇳 中文说明这是一个专为 VRoid Studio 设计的轻量级参考图悬浮工具。它无需安装，只有一个文件，双击即可运行。✨ 核心功能始终置顶：参考图永远浮在 VRoid 窗口上方。鼠标穿透 (锁定模式)：锁定后，鼠标可以直接穿透图片操作背后的模型，方便描图对照。透明度调节：右键菜单可快速调节图片透明度。自由变换：支持滚轮缩放、右下角拖拽缩放、旋转、翻转图片。零安装：基于 Windows 原生 PowerShell 编写，无需安装 Python 或其他环境。🚀 如何使用下载本项目中的 VRoidRefOverlay.bat 文件。直接双击运行 (如果 Windows 提示安全警告，请点击“仍要运行”，这是因为未签名的脚本都会有此提示)。屏幕会出现一个深色方块，右键点击它加载图片。缩放：滚动鼠标滚轮，或拖拽图片右下角。移动：按住图片任意位置拖动。锁定：右键菜单选择 Lock Image。解锁：右键点击电脑右下角托盘区的图标，选择 Unlock。🛡️ 安全声明本工具完全开源。文件本质是 Windows 批处理 (.bat) 混合 PowerShell 脚本。你可以右键点击文件选择“编辑”查看所有源代码，确保无恶意代码。<a name="english"></a>🇺🇸 English DescriptionA lightweight, always-on-top reference image overlay tool designed for VRoid Studio. No installation required, just one file.✨ FeaturesAlways on Top: Keeps your reference image above VRoid Studio.Click-Through (Lock Mode): Draw on your model "through" the image.Transparency: Adjust opacity via right-click menu.Transform: Mouse wheel zoom, drag resize, rotate, and flip.Portable: Written in native Windows PowerShell. No Python or extra libs needed.🚀 How to UseDownload VRoidRefOverlay.bat.Double-click to run.Right-click on the window to Load Image.Zoom: Use Mouse Wheel or drag the bottom-right corner.Lock: Right-click -> Lock Image to enable click-through mode.Unlock: Right-click the Tray Icon (near the system clock) -> Unlock.🛡️ SecurityThis tool is open-source.It is a mixed Batch/PowerShell script.
